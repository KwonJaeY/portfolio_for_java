PHP

웹 개발의 기초부터 PHP 언어의 핵심 개념, 웹 애플리케이션 아키텍처, 프레임워크 및 라이브러리 활용, 개발 프로세스와 협업 등 다양한 영역에서 실무 경험을 쌓아왔습니다. 웹 브라우저와 웹 서버의 상호작용, HTTP 프로토콜의 요청-응답 구조, HTML/CSS/JavaScript의 역할과 상호작용 등 웹 개발의 기초를 이해하였습니다. 이를 바탕으로 PHP 언어의 기본 문법(변수, 데이터 타입, 연산자, 제어문 등)을 익히고, 고급 문법(함수, 배열, 객체 등)을 활용하며 예외 처리, 파일 입출력, 데이터베이스 연동 등 다양한 기능을 구현할 수 있게 되었습니다.

SERVER

리눅스 운영체제의 기본 개념과 구조, 주요 서브시스템에 대한 이해를 바탕으로 실무에 필요한 기술을 익혔습니다. 이를 통해 BIND를 이용한 DNS 서버 관리, Node.js 기반 CDN 서버 관리, Apache 웹 서버 설정, API 서버 구현 등 다양한 서버 관리 경험을 쌓았습니다. DNS 서버 관리 시 서브 도메인 설정과 CDN 연동을 통해 부하를 줄이는 방법을 익혔고, Apache 웹 서버 관리에서는 도메인 포워딩, 서브 디렉토리 설정, API 연동 등의 작업을 수행하며 서버 운영에 대한 깊이 있는 이해를 갖추었습니다. 이러한 경험을 통해 리눅스 운영체제와 서버 관리에 대한 전문성을 향상시켰습니다.

DATABASE

데이터 베이스 관련한 작업 내용으로는 SQL의 JOIN과 SubQuery 기능을 활용하여 복잡한 데이터 모델링, 효율적인 데이터 조회 및 관리를 수행해왔습니다. 데이터베이스 테이블 생성과 Connection Pool 구현을 통해 데이터베이스 연결 및 자원 관리의 최적화를 경험했습니다. 이를 통해 데이터베이스 설계와 쿼리 최적화에 대한 전문성을 쌓았으며, 안정적이고 효율적인 데이터베이스 관리 능력을 보유하고 있습니다.

ANDROID

기 제작된 앱들을 사용자 피드백을 반영하여 기존 앱의 UI/UX를 개선하고 새로운 기능을 추가하는 업데이트 작업을 맡았으며, 사용자 중심의 관점에서 앱의 사용성과 접근성을 높이기 위해 노력하였고 버그 발생 시 신속한 분석과 효과적인 수정을 통해 앱의 안정성을 높였습니다 주로 관공서의 직원들의 편의를 위한 앱이였으며 이외에는 인스타그램API를 사용한 키오스크앱, 카메라와 연동하여 제작된 셀프 스튜디오의 앱 등이 있습니다.
