BACK-END

PHP를 이용한 DB 연계 및 서버 성능에 맞는 쿼리 수정, 공공데이터 API 사용, 사내 자체 CMS를 통한 개발 및 유지보수, 그누보드 개발 및 유지보수, 실시간 스트리밍 백업 프로그램(방송사 영상 백업 시스템) 개발(파이썬), YES24 서버 구매 및 기초 세팅 작업 (사내 CMS에 맞는 APM 설치 및 기본 모듈 설치) 등의 작업을 해왔습니다.

SERVER

서버 관련해서는 크게 bind를 이용한 DNS서버, node.js CDN서버, Apache 웹서버, API 서버를 관리하였습니다. DNS는 서브 도메인 설정 및 CDN을 통해 세팅 부하를 줄이고 Apache 웹서버는 도메인 포워딩, 서브 디렉토리 세팅, API 세팅 작업 등을 진행하여 서버에 대한 이해도를 높일 수 있었습니다.

WEB-FRONT

지난 2년 동안 주로 관공서 위주의 웹 프로젝트에 참여하였습니다. 이외에도 학교, 개인 사업체 등의 웹 페이지를 제작 및 관리하며 웹접근성 준수, 슬라이더 및 애니매이션, API(지도,날씨 등)등의 작업을 하였습니다. 이를 통해 웹 개발 실무 경험을 쌓았으며, 팀워크와 문제 해결 능력을 향상시킬 수 있었습니다.
